#  Django E-Commerce Project

## Description
Projet e-commerce développé avec Django permettant la gestion des produits, des utilisateurs et d’un panier simple basé sur les sessions.

---



### Authentification
- Inscription (signup)
- Connexion / Déconnexion
- Profil utilisateur
- Modification des informations personnelles

### Produits
- Liste des produits
- Détail d’un produit
- Affichage des images (MEDIA)

### Panier
- Ajouter un produit au panier
- Visualiser le panier
- Calcul du total

###  Interface
- Design Bootstrap moderne
- Responsive (mobile + desktop)

##  Run project

```bash
python manage.py migrate
python manage.py runserver
